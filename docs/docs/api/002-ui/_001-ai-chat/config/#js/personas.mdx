The `withPersonaOptions(personaOptions)` configuration method allows you to give the conversation participants a name
and a profile picture. And for the case of the bot, you can also specify a _tagline_ that will be displayed below the
bot's name in the initial screen.

> * Type: `PersonaOptions`

```tsx
interface PersonaOptions {
    bot?: BotPersona,
    user?: UserPersona,
}

interface BotPersona {
    name: string;
    picture: string | Readonly<HTMLElement>;
    tagline?: string;
}

interface UserPersona {
    name: string;
    picture: string | Readonly<HTMLElement>;
}
```

> * Usage:

```jsx
import {createAiChat, BotPersona, PersonaOptions, UserPersona} from '@nlux/core';

const botPersona: BotPersona = {
    name: 'HarryBotter',
    picture: 'https://nlux.ai/images/demos/persona-harry-botter.jpg',
    tagline: 'Mischievously Making Magic With Mirthful AI!',
};

const pictureElement = document.createElement('span');
pictureElement.innerHTML = 'ðŸ‘©';

const userPersona: UserPersona = {
    name: 'Alice',
    picture: pictureElement
}

const personaOptions: PersonaOptions = {
    bot: botPersona,
    user: userPersona
};

const aiChat = createAiChat().withPersonaOptions(personaOptions);
```
