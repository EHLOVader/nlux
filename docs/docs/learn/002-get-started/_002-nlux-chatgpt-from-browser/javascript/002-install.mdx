import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

You can start by adding _NLUX_ to your TypeScript project using your favorite package manager.
At the root of your project, run the following command:

<Tabs groupId="package" queryString>
    <TabItem value="npm" label="NPM">
        ```bash
        npm install @nlux/core @nlux/openai
        ```
    </TabItem>
    <TabItem value="yarn" label="Yarn">
        ```bash
        yarn add @nlux/core @nlux/openai
        ```
    </TabItem>
    <TabItem value="pnpm" label="PNPM">
        ```bash
        pnpm add @nlux/core @nlux/openai
        ```
    </TabItem>
</Tabs>

This will install the `@nlux/core` and `@nlux/openai` packages.
