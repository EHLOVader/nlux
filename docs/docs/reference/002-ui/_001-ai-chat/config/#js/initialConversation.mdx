The `withInitialConversation(chatItems)` method is used to enable loading of conversation history before
rendering the `AiChat` component. This is useful if you want the user to resume a conversation
from a previous session.

> * Type: `initialConversation: ChatItem[]`

```tsx
type ChatItem<AiMsg = string> = {
    role: ParticipantRole;
    message: AiMsg;
};

type ParticipantRole = 'user' | 'system' | 'ai';
```
