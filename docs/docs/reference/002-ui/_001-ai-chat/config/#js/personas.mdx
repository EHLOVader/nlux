The `withPersonaOptions(personaOptions)` configuration method allows you to give the conversation participants a name
and a profile picture. And for the case of the bot, you can also specify a _tagline_ that will be displayed below the
bot's name in the initial screen.

> * Type: `PersonaOptions`

```tsx
interface PersonaOptions {
    bot?: BotPersona,
    user?: UserPersona,
}

interface BotPersona {
    name: string;
    picture: string | Readonly<HTMLElement>;
    tagline?: string;
}

interface UserPersona {
    name: string;
    picture: string | Readonly<HTMLElement>;
}
```
