The `personaOptions` configuration object allows you to give the conversation participants a name and profile picture.
And for the case of the assistant, you can also specify a _tagline_ that will be displayed below the assistant's name in the
initial screen.

> * Type: `PersonaOptions`

```tsx
interface PersonaOptions {
    assistant?: AssistantPersona,
    user?: UserPersona,
}

interface AssistantPersona {
    name: string;
    picture: string | JSX.Element;
    tagline?: string;
}

interface UserPersona {
    name: string;
    picture: string | JSX.Element;
}
```
