# Suggest a new method

---

We are adding new methods to the AI Chat API based on user feedback. If you have a suggestion for a new method that
you would like to see added to the API, please let us know by creating a new issue on the [GitHub repository](https://github.com/nlkitai/nlux/issues)
or by reaching out to us on [Discord](https://discord.gg/SRwDmZghNB).
